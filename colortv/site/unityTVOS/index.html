UNITY TVOS

Introduction

The ReplayTV Unity Plugin is a light-weight way of including Ads into your games and apps from Unity3D using ReplayTV SDK. Currently the plugin works only on the actual device, simulators and Unity Editor will not show ads.

Current guide is prepared for Unity 5.3.1f1 version with XCode 7.2.
